# M4 — Pokédex Interactiva (Bootstrap 5 + JavaScript + Fetch)

## Descripción
Pokédex web interactiva que consume la **PokeAPI** y muestra Pokémon en tarjetas responsivas.  
Incluye navegación por tipos, modal con detalle del Pokémon y una interfaz moderna usando Bootstrap 5.

Este proyecto corresponde al **Portafolio del Módulo 4**.

---

## Funcionalidades
- Consumo de PokeAPI con `fetch` y `async/await`
- Cards responsivas con Bootstrap
- Navbar dinámica con tipos de Pokémon (datos reales desde la API)
- Modal con detalle:
  - Imagen oficial
  - Tipos
  - Altura
  - Peso
  - Descripción
- Colores reales según tipo de Pokémon
- Animación de entrada en las cards
- Diseño responsive (desktop y mobile)

---

## Tecnologías utilizadas
- HTML5
- CSS3
- Bootstrap 5
- JavaScript ES6+
- Fetch API
- PokeAPI (https://pokeapi.co/)
